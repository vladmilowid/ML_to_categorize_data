Этот репозиторий содержит проект по разработке системы автоматической категоризации текстовых данных с использованием методов машинного обучения для отдела технической поддержки интернет-эквайринга. Основная цель проекта — оптимизация процесса обработки клиентских обращений, автоматизация маршрутизации тикетов и повышение эффективности работы отдела.
